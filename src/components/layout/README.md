layout/ or auxiliary/ (Structural Modules)
These components handle the overall arrangement or scaffolding of the application or a section of a page. They provide the structure into which other components are inserted.

Role: Define the structure of the UI.

Examples: Modal.vue, Sidebar.vue, LayoutHeader.vue, TabContainer.vue.
